项目简介

sarah-jnode

sarah-jnode 是服务于星际鑫航自主研发的云操作系统 SarahOS 的微服务，目前主要处理硬件节点的警报监控，发送警报信息服务， 以及各个节点的计算力(CPU Power)，存储能力(Storage Power) 的评估服务。
